# AGENTS.md

This is the Our Studio employee handbook — a Docusaurus 3 static site. All content is in Swedish.

## Key files

- `docusaurus.config.js` — site config (title, URL, plugins, theme)
- `sidebars.js` — autogenerated from `docs/` filesystem structure
- `docs/` — all handbook pages as markdown files
- `src/css/custom.css` — global style overrides
- `static/img/` — images and favicon
- `i18n/sv/code.json` — Swedish UI translations for the search plugin

## Conventions

- Docs are served at the site root (`/`), not under `/docs/`
- Doc filenames use kebab-case (e.g. `ny-pa-jobbet.md`)
- Use `sidebar_position` in frontmatter to control ordering — the sidebar is autogenerated
- All documentation content must be written in Swedish

## Development

```sh
yarn start    # dev server with hot reload
yarn build    # production build — run this to verify changes
```

## Deployment

No CI/CD pipeline — deploy manually with `yarn deploy` (pushes to `gh-pages` branch).
